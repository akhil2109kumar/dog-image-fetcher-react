## Setup

1. **Clone the Repository:**
   ```bash
   git clone https://github.com/akhil2109kumar/dog-image-fetcher-react.git
   cd dog-image-fetcher-react
   ```
2. **Install Dependencies:**
* Node Version
  16.15.1

* yarn install command
  ```
  yarn install
  ```

* To start the server
  ```
  yarn start
  ```
